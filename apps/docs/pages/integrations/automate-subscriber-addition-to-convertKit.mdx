## Setting up ConvertKit Integration

To add the new workflow action for form submissions to send the emails to your form submitters, follow these steps:

1.  Make sure you have an account on ConvertKit if not please visit [ConvertKit](https://app.convertkit.com/users/signup) and sign up.
2.  Create the API Key by following the ConvertKit guide [Create ConvertKit API Key](https://app.convertkit.com/account_settings/advanced_settings).
3.  Log in to your dashboard and navigate to the “**Profile**” Tab.
4.  Click on Setting,“**Advanced**”.
5.  You will get the API key and API Secret

![image](/integrations/convertkit/account-connect.png)

6.  Now navigate to the “**Forms**” tab and select the form for which you want to set up ConvertKit emails workflow.
7.  Under the selected form page navigate to the “**Workflows**” Tab
8.  Under the workflows click on the “**Actions**” and select “**Add action**” this will bring up a new window.
9.  Here select the App as “**ConvertKit**” and Connection which was connected previously and "**Add Subscriber to a Form**" as an action.
10. After Selecting all the options click on “**Add**” and test the ConvertKit workflow action by submitting a test form on your website and verify the subscriber added in ConvertKit.

![image](/integrations/convertkit/convertkit-acc.png)

## Toggling the ConvertKit Workflow

Once you have added the ConvertKit action to the form’s workflow, you can toggle the workflow action by:

1.  Navigate to the the “**Workflow**” tab under the selected form
2.  Just click on the toggle switch on the action to pause or resume the workflow.

## Troubleshooting ConvertKit Workflow.

If you encounter any issues with ConvertKit workflow, such as new subscriber not being created, you can troubleshoot the issue by:

- Reconnecting the ConvertKit connection by clicking the reconnect button under the “Apps” tab and entering the “**Connection Name**” and “**API Key**”.
- Test the workflow by submitting the test submission on the form.
