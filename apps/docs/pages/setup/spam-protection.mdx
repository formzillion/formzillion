# Spam Protection

Formzillion provides the following solutions aimed at preventing spam:

- [Botpoison](https://botpoison.com/) integration
- [reCAPTCHA v2](https://www.google.com/recaptcha/about/) integration
- [reCAPTCHA v3](https://www.google.com/recaptcha/about/) integration
- [Akismet](https://akismet.com/) integration

If any of the following conditions are true, Formzillion will not save your submissions, send notifications, or decrement your submission counter:

- The submission is empty
- The spam protection verification was unsuccessful
